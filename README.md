# Quizhar

Offline-first flashcard app. Install it once — study anywhere, no WiFi needed.

## Features

- Create unlimited decks and cards
- Flip-card study mode with progress tracking
- Got it / Missed scoring with missed-card retry
- Results screen with percentage score
- All data stored locally (no account needed)
- Works offline — installable as a PWA on mobile and desktop

## Install (PWA)

1. Open the app in Chrome, Safari, or Edge
2. Tap **Add to Home Screen** (mobile) or the install icon in the address bar (desktop)
3. Done — it works offline from that point on

## Run locally

```bash
npx serve .
```

Then open `http://localhost:3000`.

## Tech

Vanilla HTML/CSS/JS + Service Worker. Zero dependencies. Zero build step.
